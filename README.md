# svelter-cropperjs

A Svelte wrapper component for [cropperjs V2](https://fengyuanchen.github.io/cropperjs/v2/).


## Installation

```shell
npm install --save svelte-cropperjs
```
or
```
yarn add svelte-cropperjs
```

# Compatibility
| Svelte Version | CropperJS Version |
| ---------- | --------------- |
| 4.x.x      | >=2.0.0         |
